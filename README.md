## React Redux Starter Kit

借鉴 [react-redux-starter-kit](https://github.com/davezuko/react-redux-starter-kit)框架 

#### 安装之前的准备 

 首先，clone 这个程序 
 	
 	 $ git clone https://github.com/davezuko/react-redux-starter-kit.git
 	 $ cd react-collapse
 
 启动步骤 
 
 	$ npm install 
 	$ npm run start 
 	
 使用技术
 
 * redux
 * es6
 * sass
 * react
 * webpack
